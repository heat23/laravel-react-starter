# Agent Instructions

This project's authoritative agent instructions live in [`CLAUDE.md`](./CLAUDE.md).

`CLAUDE.md` works for any agent that follows the AGENTS.md / CLAUDE.md convention: it contains stack versions, hard rules, decision frameworks, performance budgets, critical gotchas, and the Laravel Boost guideline subset. Path-scoped rules live in [`.claude/rules/`](./.claude/rules/) (auto-loaded when touching matching files) and Laravel ecosystem skills live in [`.claude/skills/`](./.claude/skills/).
